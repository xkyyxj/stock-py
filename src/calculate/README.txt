计算模块，通过日线信息，选择出不同的类别的股票

1. calculate_air_castle:
计算空中楼阁理论的股票，越是价格高，越是能涨

2. calculate_big_wave:
计算波动幅度大的股票

3. calculate_history_down:
计算历史低值股票，选出来并且看下后续如何走动

4. calculate_max_win:
计算最近一段时间之内，最大赢利的股票

5. calculate_quick_down:
计算短期之内快速下跌的股票，以我的经验，这种票子一般都会经历一波V型反转，快速下跌快速上升。

6. calculate_down_then_flow:
计算下跌后走平的股票，这种票子如果平的时间比较久了的话，可能会经历一波反弹（一个人也不可能一直衰的理论……）
