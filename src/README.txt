此处总说明一下各个表的作用
下面每张表里面都会有一个字段叫做几日盈利，可以做一个程序计算一下几日盈利，到时候可以汇总下看下结果如何
（如果结果一直不理想的话，可以调整选股策略，或者看下哪个策略比较好一点，而后也可以动态添加新的策略（当然要写代码））
核心表：
1. curr_hold：当前持仓的情况
2. daily_concern: 每日关注的股票
3. wait_select: 待选股票，程序分析出来的待选股票就放到这里面，每天数量不要太多，不然的话，会看不过来的啊

分析结果表：
1. history_down： 历史低值，此表里面数据表明股价当时处于历史低价
3. box_style：盒式模型，表明处于震荡当中
4. air_castle： 空中楼阁，股价可能越来越高——博傻博傻
5. down_then_flow： 下降然后走平的股票，可能会有忽然的爆发

选股策略选出来的股票：
1. short_time_in_time： 短期选股选出来的股票,只存储当天选择出来的股票，照理每天交易日结束后就清空掉
2. short_time_history: short_time_in_time的历史信息，看下什么情况
3. long_time_in_time: 长期选股选出来的股票,只存储当天选择出来的股票，照理每天交易日结束后就清空掉
4. long_time_history: long_time_in_time的历史信息，看下什么情况
